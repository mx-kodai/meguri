## 4. 主要スポット

### 五百羅漢（長慶寺）
- 535体の石仏（1799-1849、50年かけて完成）
- 越中の豪商・黒木善次郎が寄進
- 拝観無料、所要約20-30分
- 立山連峰をバックに並ぶ構図が秀逸

### 呉羽山展望台
- 標高約70m、24時間無料
- 日本夜景遺産認定
- 立山連峰+北陸新幹線の共演が撮影可能

### 民俗民芸村
- 9つの博物館施設群
- 篁牛人記念美術館（渇筆の水墨画）
- 茶室円山庵（抹茶+和菓子 310円）

### 杉谷古墳群
- 11基（四隅突出型墳丘墓含む）
- 出雲との交流を示す遺構

### 北代縄文広場
- 約4,000年前の大集落跡
- 竪穴住居・高床建物を実物大復元

### 呉羽のグルメ・カフェ
- 円山庵（抹茶310円、民芸村内）
- レストラン青い丘（暖炉、眺望抜群）
- 空の森 珈琲焙煎所（自家焙煎、Wi-Fi完備）
- dill.（2024年11月オープン、絶景）
- マリーマリー（家庭薬膳、展望台近く）

## 6. ベンチマーク調査（売れるツアーサイトの共通要素）

### 参考にすべきサイトTOP3
1. **KUMANO TRAVEL** — 地方体験ツアーの予約導線、口コミ(4.87/188件)、語り部紹介
2. **登山ガイドソレイユ** — 少人数ガイドの不安払拭コピーが秀逸
3. **PINK** — 大人向け色彩、「催行決定」「満員御礼」のソーシャルプルーフ

### 40代以降に刺さる5つのコピーパターン
1. 感情的問いかけ型（「旅に出ませんか？」）
2. 体験の具体化型（数字・データで説得）
3. 不安払拭型（「少人数」「初心者にも優しい」「一人でも安心」）
4. 権威・品質訴求型（受賞歴、メディアブランド）
5. 日常と非日常の架け橋型（「上質な非日常」）

### 信頼構築5層
1. ブランド権威
2. ガイドの人格（顔写真+資格+人柄）
3. 数値的証拠（口コミ数、評価点）
4. ステータス表示（催行決定、残りわずか）
5. 安心の仕組み（キャンセルポリシー、保険）

### 競合料金帯
- 半日散策ガイド: 3,000〜5,000円（高岡まち歩き、五箇山）
- 半日体験ツアー: 6,000〜13,000円（ちょこたび富山）
- 1泊2日パッケージ: 21,000〜48,000円（黒部峡谷プレミアム）
- → 呉羽山1泊2日: **25,000〜35,000円が現実的レンジ**

### 呉羽山の差別化（競合にない強み）
1. 富山駅から10分（立山・黒部・五箇山は遠い）
2. 標高差70mで体力に優しい
3. 歴史資産がコンパクトに集中
4. 公式観光サイトに呉羽山コースなし = 穴場
5. くれはマイスター = 「語り部」型のストーリーテリング

### クラブツーリズムから学ぶリピート設計
- シリーズ化（春夏秋冬4回）
- 完歩認定証・記念バッジ
- 1名参加歓迎の明示
- 歩いた後の美食が動機
